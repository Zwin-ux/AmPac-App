# Demo Account Information for App Store Review

## Demo Mode Access

The AmPac Capital app includes a **Demo Mode** button on the landing screen that provides full access to all app features without requiring account creation.

### How to Access Demo Mode:
1. Launch the app
2. On the landing screen, tap the **"Demo Mode"** button (orange button at the bottom)
3. This will automatically log you in as a demo user with full access to all features
